## 📌 Overview

A modern **YouTube Clone** built with a scalable architecture using **React + Node.js**, powered by the **YouTube Data API v3**.

This app replicates core YouTube features like:

* Video browsing
* Search functionality
* Video playback

> ⚡ API-driven | 🎯 Clean UI | 🚀 Production-ready structure

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

```bash id="cmd1"
git clone https://github.com/your-username/youtube-clone.git
cd youtube-clone
```

---

### 2️⃣ Install Dependencies

```bash id="cmd2"
cd client && npm install
cd ../server && npm install
```

---

### 3️⃣ Setup Environment Variables

Create `.env` inside `/server`

```id="env123"
YOUTUBE_API_KEY=your_api_key_here
PORT=5000
```

---

### 4️⃣ Run Project

```bash id="cmd3"
# Start backend
cd server
npm run dev

# Start frontend
cd client
npm start
```
